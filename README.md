# Problem Solutions 2-8

This repository contains Python implementations for the requested problems (2–8).

Files:
- `digit_sum_and_coprime_sum.py`: earlier solutions for digit-sum and coprime sum.
- `problems_2_to_8.py`: implementations for problems 2..8 with a small CLI/tests at the bottom.

Quick run (from repository root):

```bash
python -m problems_2_to_8
```
